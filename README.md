# Weyl Spectrum

This repository contains computational and expository resources on the Weyl spectrum, including recursive constructions of Salem and Pisot numbers and related data tables.

## Data files

- `Primitive First 1688.csv`: complete list of 1688 Salem numbers currently included in the table.
- `Primitive Level1.csv`: newly emerging primitive Salem numbers at s0-level 1.
- `Primitive Level2.csv`: newly emerging primitive Salem numbers at s0-level 2.
- `Primitive Level3.csv`: newly emerging primitive Salem numbers at s0-level 3.
- `Primitive Level 4.csv`: newly emerging primitive Salem numbers at s0-level 4.
- `Primitive Level 5.csv`: newly emerging primitive Salem numbers at s0-level 5.

The column `s0_level` refers to the level with respect to the generator s0.
